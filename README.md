# ABOUT

- A lightweight and minimal **waitlist website** that collects user info for `Trust Vault Waitlist` and stores them directly in **Google Sheets** no backend server required.
- `Trust Vault` is a blockchain project to help users secure their crypto.

## Tech Stack

- **Frontend:** HTML, CSS, Vanilla JavaScript  
- **Backend (Serverless):** Google Apps Script  
- **Database:** Google Sheets
